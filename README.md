# code-review-cli

This tool support macbooks only.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.4. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.


## Prerequisites

Setup BB_TOKEN env variable with your BitBucket Personal Access Token. 
Setup ANTHROPIC_API_KEY env variable with your Anthropic API key. 
Setup PATH_TO_CLAUDE env variable with the path to your Claude Code executable file.

Later this will be replaced with config file.
